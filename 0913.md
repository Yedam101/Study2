### 1193 분수찾기
```python

n = int(input())

dp = [0]*100001
dp[1], dp[2] = 1, 2

if n < 3:
    print(f'1/{n}')
else:
    for i in range(3, 100001):
        dp[i] = dp[i-1] + (i-1)
        if dp[i-1] <= n < dp[i]:
            a = n-dp[i-1]
            if (i-1) % 2 == 0:
                print(f'{1+a}/{i-1-a}')
            else:
                print(f'{i-1-a}/{1+a}')  

```
